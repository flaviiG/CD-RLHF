# Minimal dependency list for running CD-RLHF on a single-GPU vast.ai box.
#
# Why this is short: the original requirements.txt was a `pip freeze` that
# captured dozens of unused transitive deps (pyext, torch-struct, unitxt,
# benepar, huspacy, langid, contractions, streamlit, pdfkit, Wand, ...) — most
# of which are not imported anywhere in the codebase and several of which break
# on Python 3.11+ (e.g. pyext calls inspect.getargspec, removed in 3.11).
#
# This file lists only what is actually imported by:
#   - applications/DeepSpeed-Chat/dschat
#   - applications/DeepSpeed-Chat/training/step{1,2,3}/*.py
#   - evaluation/*.py
# Pip will pull transitives automatically.
#
# Install torch separately first (CUDA-matched wheel) — see setup_vastai.sh.
